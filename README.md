# Proyek Website 7Icons

Website company profile sederhana untuk brand 7Icons.
Repository ini berisi source code website statis beserta admin panel sederhana
yang dibuat menggunakan HTML, CSS, dan JavaScript.

## Fitur
- Landing page modern
- Admin panel sederhana
- Layout responsif
- Struktur folder rapi

## Struktur Folder
- admin-panel/ → halaman admin
- proyek website/ → halaman utama website
- Proyek Template/ → template awal project

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript

## Tujuan Project
Project ini dibuat untuk pembelajaran dan portofolio pengembangan web.
